Internal Mechanisms — How each optimiser computes the update

SGD
θ ← θ − η · ∇L(θ)
η = 0.001
Raw gradient, scaled by a fixed LR. No history, no adaptation. Every parameter uses the same step size regardless of its gradient magnitude.
No memory · Fixed LR
SGD + Momentum
v ← β·v + η·∇L(θ) θ ← θ − v
η = 0.001  ·  β = 0.9
Velocity accumulates past gradients (90% retained per step). Consistent directions build speed; oscillating directions cancel — dampening ravine zig-zag.
Velocity memory · Fixed LR
RMSprop
s ← ρ·s + (1−ρ)·g² θ ← θ − η · g / √(s+ε)
η = 0.001  ·  ρ = 0.9
Divides each gradient by its own recent RMS magnitude. Large-gradient params get smaller steps; small-gradient params get larger steps. Per-parameter LR scaling with no momentum.
No momentum · Per-param LR
Adam
m ← β₁·m + (1−β₁)·g v ← β₂·v + (1−β₂)·g² θ ← θ − η · m̂ / (√v̂ + ε)
β₁=0.9  ·  β₂=0.999  ·  η=0.001
Combines momentum (m) and per-param LR scaling (v). Bias correction (m̂, v̂) prevents cold-start underestimation in early epochs. Best of SGD+Momentum and RMSprop.
Momentum + Per-param LR + Bias correction
